# AI PR Reviewer

Automatically review GitHub pull requests using AI.

## Features

- Detect potential bugs
- Suggest code improvements
- Identify security issues
- Assist maintainers with pull request reviews

## How it works

1. A pull request is opened
2. GitHub Actions extracts the code diff
3. AI analyzes the changes
4. Suggestions are generated

## Setup

Add your OPENAI_API_KEY to repository secrets.

## Use cases

- open-source maintainers
- automated code review
- CI workflows
