# Contributing to AI PR Reviewer

Thanks for your interest in contributing to AI PR Reviewer.

This project aims to help open-source maintainers review pull requests faster with AI-assisted feedback. Contributions that improve reliability, review quality, GitHub integration, safety, and developer experience are welcome.

## Ways to contribute

You can help by:

- reporting bugs
- suggesting features
- improving review quality and prompts
- improving GitHub Action integration
- adding tests
- improving documentation
- proposing safer defaults for OSS usage

## Before you start

Please:

1. Search existing issues before opening a new one.
2. Open an issue first for large changes.
3. Keep pull requests focused and small when possible.
